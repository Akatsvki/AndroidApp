View Records Function

This function is paired with the user profile since the it will be the best place to show all user records. There are 3 sections to fully distinguish each record from one another. Each sections comes with data retrieved from the database and then displayed. The (1) Due Today Section are notification boxed the user will receive throughout the day. (2) Similar to the former, the All Reminders Section are all data retrieved through the Add Medicine and Set Reminder function and displayed in a notification bubble. This part will display both reminder and medicine details. (3) The History will include all the expired reminders as well as the finished medication count.


The UI comes with a scrollable feature to allow the users to navigate through each section. Moreover, the user is given autonomy to edit and confirm nbotification boxes. Apart from this, the user can also edit and set his profile account through the edit button.
